MeetEval alignment visualization tool

This tool visualizes word sequence alignments to facilitate spotting of errors in conversational ASR transcription

Examples

Poster example

The example displayed on the poster presented at ICASSP 2024 Show&Tell.

LibriCSS tcpWER

A high-overlap (OV40) example from the LibriCSS dataset.

LibriCSS CSS tcORC-WER

A high-overlap (OV40) example from the LibriCSS dataset, separated with a Continuous Speech Separation (CSS) pipeline and evaluated with the speaker agnostic tcORC-WER.

DiPCo tcpWER

Error analysis on a DiPCo session processed by Paderborn University's submission to the 7th CHiME challenge.

Side-by-side System Comparison

Two synchronized visualizations to compare different systems. Here: NTT's and Paderborn University's submissions to the 7th CHiME challenge (DiPCo dataset).

Side-by-side tcpWER vs. cpWER

Comparison of the tcp alignment (with time constraint) with the cp alignment (without time constraint).

JupyterLite Notebook

The JupyterLite notebook that generates the visualization in the browser. Run it with you own data, no installation required!

Session Browser

Browse results across different sessions.
